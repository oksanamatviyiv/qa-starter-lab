# QA Starter Lab

This repository contains a Pytest-based test suite for both API and UI testing.

## Project Structure

- **src/**: Contains application source code modules for API client and UI pages.
- **tests/**: Holds Pytest test cases organized by api, ui_playwrite, and ui_selenium.
- **reports/**: Stores test reports generated by the test runs.

## Running Tests

Activate the virtual environment and run:

```bash
pytest -q
```

Reports are generated in the `reports/` directory.
